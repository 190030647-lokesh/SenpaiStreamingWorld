How to Fix 401 Errors:

  1. If you see "Demo Mode", the database is not connected. Contact admin to check Railway MySQL service.
  2. If you registered in demo mode, your credentials are NOT in the database. You need to register again when database is connected.
  3. Clear your browser's localStorage and register again with the database connected.
  4. After fixing, login with your credentials and they will be saved properly.
